Quick Start Guide

  1. Download the GO6 API Wrapper executable from our Discord server.
  2. Run the executable. It will automatically start the local API server (typically on port 5000).
  3. The API will be available at http://localhost:5000/ or your machine's local IP.
  4. Begin making requests to the documented endpoints below using your preferred language or tool.

Join a Discord server using an invite code with the specified token. Proxies are automatically rotated if multiple are configured.

Request Body (JSON)

ParameterTypeDescription
token *stringDiscord account token.
invite *stringDiscord invite code (e.g., "discord").
proxy *stringProxy (ip:port:user:pass or ip:port).

Example Request

Local API Call
curl -X POST "http://localhost:5000/join" \
-H "Content-Type: application/json" \
-d '{
  "token": "YOUR_DISCORD_TOKEN",
  "invite": "TARGET_INVITE_CODE",
  "proxy": "ip:port:user:pass"
}'
fetch('http://localhost:5000/join', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    token: 'YOUR_DISCORD_TOKEN',
    invite: 'TARGET_INVITE_CODE',
    proxy: 'ip:port:user:pass'
  })
})
.then(response => response.json())
.then(data => console.log(data));
import requests

payload = {
  "token": "YOUR_DISCORD_TOKEN",
  "invite": "TARGET_INVITE_CODE",
  "proxy": "ip:port:user:pass"
}
response = requests.post("http://localhost:5000/join", json=payload)
print(response.json())

Example Responses

JSON Response
{
  "status": "success",
  "message": "Successfully joined server.",
  "server_id": "123456789012345678",
  "response_time": "3ms"
}
{

  "response_time": "1ms"

Send a message to a specified Discord channel using a token.

Request Body (JSON)

ParameterTypeDescription
token *stringDiscord account token.
channel_id *stringID of the target channel.
content *stringThe message content to send.
proxy *stringProxy (ip:port:user:pass or ip:port).

Example Request

Local API Call
curl -X POST "http://localhost:5000/message" \

-d '{
  "token": "YOUR_DISCORD_TOKEN",
